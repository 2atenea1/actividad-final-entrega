# Datos para las bases de datos

## MySQL - Pokemon

Archivo:

```text
database-scripts/mysql-pokemon.sql
```

Incluye:

- Base `pokemon_db`.
- Tabla `pokemons`.
- 12 pokemones.

## PostgreSQL - Naruto

Archivo:

```text
database-scripts/postgresql-naruto.sql
```

Incluye:

- Tabla `personajes`.
- 12 personajes de Naruto.

## MongoDB - Dragon Ball

Para pegar documentos en Atlas:

```text
database-scripts/mongodb-dragonball-documents.json
```

Para ejecutar como script:

```text
database-scripts/mongodb-dragonball.js
```

Incluye:

- Base `dragonball_db`.
- Coleccion `guerreros`.
- 12 personajes de Dragon Ball.
